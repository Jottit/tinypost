# Jottit

Jottit is a radically simple blogging platform. You sign up, you get a blog at yourname.jottit.pub, and you start writing. That's it.

Originally co-created with [Aaron Swartz](https://en.wikipedia.org/wiki/Aaron_Swartz) in 2007. Now being rebuilt as open infrastructure for web publishing.

## Status

Early development. See [jottit.pub](https://jottit.pub).
